Vector Retriever with LangChain, Groq, and HuggingFace
This project demonstrates how to use LangChain with Groq API and HuggingFace models to generate vector embeddings and interact with language models.

🛠 Requirements
Install all the dependencies using:

bash
Copy
Edit
pip install -r requirements.txt
🔐 Environment Variables
Create a .env file in the project root and add your API keys:

env
Copy
Edit
GROQ_API_KEY=your_groq_api_key_here
HF_KEY=your_huggingface_api_key_here
Replace the placeholder values with your actual API keys.
